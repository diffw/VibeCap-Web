## Code Patterns

### Basic Test Structure
```swift
import XCTest
import SnapshotTesting
@testable import {PROJECT_NAME}

final class SomeViewSnapshotTests: XCTestCase {
    func testViewLight() {
        let view = SomeView()
            .environment(\.colorScheme, .light)
            .frame(width: 400, height: 300)
        assertSnapshot(of: view, as: .image)
    }

    func testViewDark() {
        let view = SomeView()
            .environment(\.colorScheme, .dark)
            .frame(width: 400, height: 300)

}
```

## Platform-Specific Considerations

- macOS snapshots render at the host display's scale factor (typically 2x on Retina). Be consistent across CI and local.
- Use `.image` strategy for SwiftUI views with explicit `.frame()` to ensure consistent size.
- Test both light and dark color schemes.
- Avoid network/file IO in snapshot tests; bundle minimal fixtures.
