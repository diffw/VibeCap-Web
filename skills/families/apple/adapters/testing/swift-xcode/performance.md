## Platform-Specific Considerations

- Focus on hot paths: core processing pipelines, data transformations, heavy computations.
- macOS has more memory and CPU headroom than iOS — set appropriate thresholds.
- Keep measurement blocks minimal; avoid UI or network inside performance tests.
- Use `XCTMemoryMetric()` to catch memory leaks in processing pipelines.
