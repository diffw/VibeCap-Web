## 2. 数据模型 (Data Model)

### 2.1 本地模型 (SwiftData)

```swift
@Model
final class Project {
    @Attribute(.unique) var id: UUID
    var name: String
    var color: String  // HEX
    var isArchived: Bool = false
    var createdAt: Date
    @Relationship(deleteRule: .cascade) var tasks: [Task] = []
}

@Model
final class Task {
    @Attribute(.unique) var id: UUID
    var title: String
    var taskDescription: String?
    var status: TaskStatus = .todo
    var priority: TaskPriority = .medium
    var dueDate: Date?
    var completedAt: Date?
    var position: Int
    var createdAt: Date
    var project: Project?
}

enum TaskStatus: String, Codable, CaseIterable {
    case todo, inProgress = "in_progress", done
}

enum TaskPriority: String, Codable, CaseIterable {
    case low, medium, high, urgent
}
```

```
│       [  注册  ]            │

│  ───────── 或 ─────────    │
│                             │
│  [  使用 Apple 登录]       │

```

**UI组件**:

| 组件 | 类型 | 属性 |
|------|------|------|
| 邮箱输入 | TextField | keyboardType: .emailAddress, textContentType: .emailAddress |
| 密码输入 | SecureField | textContentType: .newPassword |
| 显示密码 | Button | 切换SecureField/TextField |
| 注册按钮 | Button | .buttonStyle(.borderedProminent) |
| Apple登录 | SignInWithAppleButton | ASAuthorizationAppleIDButton |

**校验规则**:

| 字段 | 规则 | 错误提示 |
|------|------|----------|
| 邮箱 | 必填，有效格式 | "请输入有效的邮箱地址" |
| 密码 | 至少8位，含大小写和数字 | "密码至少8位，需包含大小写字母和数字" |

**交互规则**:

| 触发 | 行为 | 反馈 |
|------|------|------|
| 点击注册 | 校验 → 调用API | 按钮ProgressView |
| 注册成功 | 发送验证邮件 | Alert提示查收邮件 |
| 注册失败 | 显示错误 | 对应字段标红或Alert |

**异常处理**:

| 异常 | 响应 |
|------|------|
| 邮箱已注册 | Alert: "该邮箱已注册"，提供登录按钮 |
| 网络错误 | Alert: "网络连接失败，请重试" |

### 3.10 推送通知

**通知类型**:

| 类型 | 触发条件 | 标题 | 内容 |
|------|----------|------|------|
| 任务到期 | 截止日期当天9:00 | 任务即将到期 | "「{标题}」今天到期" |

- 任务相关 → Deep Link打开任务详情
- 摘要 → 打开首页

**权限请求时机**: 用户首次启用通知功能时
