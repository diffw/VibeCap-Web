## 4. 功能模块详情 (Feature Specifications)

### 4.1 App启动

**启动行为设置**:

| 设置 | 行为 |
|------|------|
| "显示主窗口" | 启动后显示主窗口 |
| "仅菜单栏" | 启动后仅显示菜单栏图标 |
| "恢复上次状态" | 记住上次退出时的状态 |

**登录项启动**:
- 可在偏好设置中开启"登录时启动"
- 登录项启动时默认仅显示菜单栏

```swift
// AppDelegate.swift
func applicationDidFinishLaunching(_ notification: Notification) {
    setupMenuBar()
    
    if launchAtLogin && !launchedByUser {
        // 登录项启动，仅显示菜单栏
    } else {
        showMainWindow()
    }
}
```

### 4.4 快速添加 (Quick Add)

**触发方式**:
1. 菜单栏Popover中的输入框
2. 全局快捷键（默认：Cmd+Shift+A）
3. 主窗口 Cmd+N

**全局快速添加窗口**:
```
┌───────────────────────────────────────────────────┐
│ 🔍  添加任务...                              Esc  │
├───────────────────────────────────────────────────┤
│  最近项目:  [工作]  [个人]  [收件箱]              │
└───────────────────────────────────────────────────┘
```

**位置**: 屏幕中央偏上

**快捷语法**: 同Browser Extension
- `!1` 紧急 `!2` 高 `!3` 中 `!4` 低
- `#项目名` 指定项目
- `@today` `@tomorrow` `@2024-01-20` 截止日期

**行为**:
- 打开时自动聚焦输入框
- Enter 创建并关闭
- Esc 取消
- 支持从任意App唤起（需辅助功能权限）

## 7. 数据同步

**同步策略**:
- 本地优先，后台同步
- 冲突解决：后写入优先
- 离线支持：本地存储，联网后同步

**同步状态指示**:
- 菜单栏图标动画（同步中）
- 主窗口底部状态栏

## 8. 错误处理

| 错误 | 处理 |
